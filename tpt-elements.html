More shortcuts:
[M]                     switch particle debug item
[X][C]                  show debug HUD as particle's ctype
[X][Shift+C]            show debug HUD as coordinate
[X][D]                  show debug HUD as decoration color
[X][Shift+D]            show debug HUD as secondary decoration color
[X][E]                  show debug HUD as emap
[X][F]                  show debug HUD as flags
[X][L]                  show debug HUD as life
[X][P]                  show debug HUD as pavg
[X][Shift+P]            show/hide sim_max_pressure
[X][T]                  show debug HUD as particle's type
[X][Shift+T]            show debug HUD as particle's tmp
[X][Alt+T]              switch temperature as C/F/K
[X][V]                  show debug HUD as particle's velocity
[X][Shift+V]            show debug HUD as air velocity
[X][X]                  switch hexadecimal mode
[X][+]                  increase precision in debug HUD
[X][-]                  decrease precision in debug HUD

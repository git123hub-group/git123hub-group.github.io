Sign tool usage:
  "{aheat}"     : shows ambient heat in sign
  "{b:dt}"      : clickable button
  "{c:n|dt}"    : linking to other saves
  "{p}"         : shows pressure in sign
  "{s:t|dt}"    : search online saves
  "{t}"         : shows temperature in sign
  "{t:n|dt}"    : linking to forum threads
  "{vx}"        : shows air velocity x in sign
  "{vy}"        : shows air velocity y in sign
  Particle debug sign:
    "{.xxx}"        : shows value as integer
    "{.xxx:f}"      : shows value as float
    "{.xxx:t}"      : shows value as particle type
    "{.xxx:x}"      : shows value as hexadecimal
    "{.xxx:xt}"     : shows value as extended particle type
    "{.ctype}"      : shows particle's ctype
      "{.ctype:t}"
    "{.i}"          : shows particle's ID
    "{.life}"       : shows particle's life
    "{.pavg0}"      : shows particle's pavg[0]
      "{.pavg0:f}"
    "{.pavg1}"      : shows particle's pavg[1]
      "{.pavg1:f}"
    "{.tmp}"        : shows particle's tmp
      "{.tmp:f}"
      "{.tmp:t}"
    "{.tmp2}"       : shows particle's tmp2
    "{.tmp3}"       : shows particle's tmp3
    "{.tmp4}"       : shows particle's tmp4
    "{.type}"       : shows particle's type
      "{.type:t}"
    "{.vx}"         : shows particle's vx
      "{.vx:f}"
    "{.vy}"         : shows particle's vy
      "{.vy:f}"

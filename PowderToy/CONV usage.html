Element number 85 (CONV) usage:
  .tmp = element to be converted (particle type source)
    if .tmp = 0, then converting anything (without indestructible element (e.g. DMND is an indestructible element)).
    if .tmp = 28 (DMND), then do nothing.
  .ctype = converted element (particle type destination)

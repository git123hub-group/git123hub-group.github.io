TPT 配置

Simulation properties:
   CELL : The size of the pressure, gravity, and wall maps (4)
   Walls properties:
      sim->bmap[y/CELL][x/CELL] : block-map at (x, y)
      sim->emap[y/CELL][x/CELL] : electronics-map at (x, y)
      sim->fvx[y/CELL][x/CELL] : fan velocity x at (x, y)
      sim->fvy[y/CELL][x/CELL] : fan velocity y at (x, y)
      sim->breakable_wall_count : the numbers of breakable wall
      sim->sim_max_pressure : maximum pressure (include negative pressure) for breakable wall
   Particles properties:
      sim->parts[i] : the particle that has index i [粒子属性表]
      sim->pmap[y][x] : particle-map at (x, y) [粒子映射表]
      sim->photons[y][x] : photon-map at (x, y) [能量粒子映射表]
   Air properties:
      sim->pv[y/CELL][x/CELL] : the pressure at (x, y) [气压表]
      sim->vx[y/CELL][x/CELL] : the air velocity x at (x, y)
      sim->vy[y/CELL][x/CELL] : the air velocity y at (x, y)
      sim->hv[y/CELL][x/CELL] : the ambient heat at (x, y) [环境热表]
   Gravity properties:
      sim->gravmap[y/CELL][x/CELL] : the gravity-map at (x, y)
      sim->gravx[y/CELL][x/CELL] : the gravity velocity x at (x, y)
      sim->gravy[y/CELL][x/CELL] : the gravity velocity y at (x, y)
   Air update properties:
      sim->air->pv : input for pressure
      sim->air->opv : output for pressure
      sim->air->vx : input for air velocity x
      sim->air->ovx : output for air velocity x
      sim->air->vy : input for air velocity y
      sim->air->ovy : output for air velocity y
      sim->air->hv : input for ambient heat
      sim->air->ohv : output for ambient heat
   
Single particle properties:
   .type : particle's current type [主要粒子类型]
      Can change by C++ code: sim->part_change_type(id, x, y, type)
   .life : particle's life
   .ctype : particle's previous type [次要粒子类型, 其实可以是任意类型]
   .x, .y : particle's coordinate (using float, not integer)
   .vx, .vy : particle's velocity
   .temp : particle's temperature
   .tmp, .tmp2, etc. : a value which can be used for anything.
   .dcolour : particle's decoration color
   .cdcolour : particle's previous decoration color
      Can change by Lua code: simulation.secondaryDeco, not simulation.partProperty
      Can't change by console: "!set cdcolour", or PROP (Property tool)
   .pavg: Mainly used for pressure or storing properties of particles inside PIPE
   Useless particle properties:
      .ctype isn't a particle in Powder Toy
      .clife : ctype's life?
      .cctype : ctype's ctype?
      .ctemp : ctype's temp?
      .ctmp : ctype's tmp?
      .ctmp2 : ctype's tmp2?
      .cpavg : ctype's pavg?
      .cclife : ctype's clife?
      .ccctype : ctype's cctype?
      .cctemp : ctype's ctemp?
      .ccclife : ctype's cclife?

"Element.h":
   IPL = Impossibly Low Pressure
   IPH = Impossibly High Pressure
   ITL = Impossibly Low Temperature
   ITH = Impossibly High Temperature
   NT  = No transition (-1)
   ST  = Special transition (PT_NUM)

"Elements.h":
   R_TEMP     = Room temperature (22 C)
   MAX_TEMP   = Maximum temperature (9999 K)
   MIN_TEMP   = Minimum temperature (0 K)
   O_MAX_TEMP = Maximum temperature (old) (3500 C)
   O_MIN_TEMP = Minimum temperature (old) (-273 C)
   1st properties:
      TYPE_PART               = Powder type (1)
      TYPE_LIQUID             = Liquid type (2)
      TYPE_PART               = Solid type (4)
      TYPE_LIQUID             = Gas type (8)
      TYPE_ENERGY             = Energy type (16)
      STATE_FLAGS             = Any type (31)
      PROP_CONDUCTS           = Conducts electricity (32)
      PROP_BLACK              = Unused (64)
      Neutrons interaction:
         Reflects neutrons is default
         PROP_NEUTPENETRATE   = Penetrated by neutrons (128)
         PROP_NEUTABSORB      = Absorbs neutrons (256)
         PROP_NEUTPASS        = Neutrons pass through (512)
      PROP_DEADLY             = Deadly for stickman (1024)
      PROP_HOT_GLOW           = Hot Metal Glow (2048)
      PROP_LIFE               = Unused (4096)
      PROP_RADIOACTIVE        = Radioactive (8192)
      properties for .life:
         PROP_LIFE_DEC        = .life decreases by one every frame if > 0 (2^14)
         PROP_LIFE_KILL       = Kill when .life value is <= 0 (2^15)
         PROP_LIFE_KILL_DEC   = Kill when .life value is decremented to <= 0 (2^16)
      PROP_SPARKSETTLE        = Allow Sparks/Embers to settle (2^17)
      PROP_NOAMBHEAT          = Don't transfer or receive heat from ambient heat. (2^18)
      PROP_DRAWONCTYPE        = Set its ctype to another element if the element is drawn upon it. (2^19)
      PROP_NOCTYPEDRAW        = When this element is drawn upon with, do not set ctype (2^20)
      PROP_NOSLOWDOWN         = Non-official property (2^25)
      PROP_TRANSPARENT        = Non-official property (2^26)
      PROP_UNLIMSTACKING      = Non-official property (2^27)
      PROP_INSULATED          = Non-official property (2^28)
      PROP_CONDUCTS_SPEC      = Non-official property (2^29)
      PROP_NO_NBHL_GEN        = Non-official property (2^30)

   2nd properties:
      PROP_DEBUG_USE_TMP2     = 1
      PROP_ELEC_HEATING       = 2
      PROP_NOWAVELENGTHS      = 4
      PROP_CTYPE_INTG         = 8
      PROP_CTYPE_WAVEL        = 16
      PROP_CTYPE_SPEC         = 32
      PROP_ENERGY_PART        = 64
      PROP_UNBREAKABLECLONE   = 16384
      PROP_NODESTRUCT         = Indestructible property (32768)
      PROP_CLONE              = 65536
      PROP_DRAWONCTYPE        = 2^19
      PROP_NOSLOWDOWN         = 2^25
      PROP_INVISIBLE          = Acts like INVS (2^26)

   .flags properties:
      FLAG_STAGNANT   = stagnant (1)
      FLAG_SKIPMOVE   = skip movement for one frame (2)
      FLAG_WATEREQUAL = if a liquid was already checked during equalization (4)
      FLAG_MOVABLE    = compatibility with moving SPNG (8)
      FLAG_PHOTDECO   = compatibility with decorated PHOT (8)

   BOUNDS_CHECK = bounds check (ignored)
   REAL_BOUNDS_CHECK(x, y) = sim->InBounds(x, y)
   OLD_PT_WIND = Unused type (147)
   PT_NUM = 256

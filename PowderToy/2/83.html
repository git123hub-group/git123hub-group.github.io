Element number 83 (STOR) usage:
  .ctype = store that type of particle.
    if .ctype = 0, then can store any particles.
  .tmp = absorbed element's .type value
  .temp = absorbed element's .temp value
  .tmp2 = absorbed element's .life value
  .tmp3 = absorbed element's .tmp value
  .tmp4 = absorbed element's .ctype value
  .pavg[0] = absorbed element's .tmp2 value
  .pavg[1] = absorbed element's .tmp3 value
  .cdcolour = absorbed element's .dcolour value
